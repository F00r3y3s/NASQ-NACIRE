# Logging and Handoffs

## Required Files
- formal spec files
- research notes
- decision ledger
- implementation rules
- docs-grounding checklist
- execution status

## Purpose
These files prevent context rot across:
- long threads
- fresh conversations
- sub-agent work
- pauses between phases

## Handoff Rule
Every new thread should be able to start from the markdown package without rereading raw source artifacts unless visual verification is needed.
